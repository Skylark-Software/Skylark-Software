Janua

Enhanced Apache Guacamole for modern Wayland remote desktop

Janua solves compatibility issues between Guacamole and GNOME Remote Desktop/KDE KRdp by:

  • Building guacd against FreeRDP 3.10.3
  • Patching guacamole-server to advertise RDPSND version 8
  • Enabling H.264/AVC codecs for RDPGFX (required by KDE KRdp)
